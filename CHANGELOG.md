# Changelog

모든 중요한 변경 사항이 이 파일에 기록됩니다.

---

## v18 (2024-11-14)

### ✨ Added
- **카드 추가 시 카테고리 생성 기능**
  - 카드 다이얼로그에서 직접 새 카테고리 추가 가능
  - "+" 버튼 클릭으로 카테고리 생성 폼 표시
  - 생성된 카테고리는 좌측 사이드바에 자동으로 추가됨
  - 카테고리 이름, 아이콘, 배경색 설정 가능

### 🐛 Fixed
- **CSV 한글 인코딩 문제 해결**
  - UTF-8 BOM 처리 로직 추가
  - Excel에서 내보낸 CSV 파일의 한글이 깨지지 않고 정상적으로 가져와짐
  - CSV 파일 읽기 시 BOM 자동 제거

### 🔧 Changed
- 버전 v17 → v18로 업데이트

## v17 (2024-11-14)

### ✨ Added
- **카테고리 편집 기능**
  - 카테고리 목록에서 편집 버튼 추가
  - 카테고리 이름, 아이콘, 배경색 수정 가능
  
- **직접 텍스트 입력**
  - 문장 만들기 바에서 키보드로 직접 입력 가능
  - 카드 선택과 키보드 입력을 자유롭게 혼합 사용
  - 입력한 텍스트도 TTS로 음성 출력

- **"사랑해요" 카드 추가**
  - 감정 카테고리에 "사랑해요" 카드 추가
  - 커스텀 이미지 포함
  - 기본 카드로 제공

### 🎨 Improved
- **태블릿 최적화**
  - 가로 모드에서 한 화면에 모든 요소 표시
  - 앱바 + 카드 그리드 + 문장 바를 동시에 표시
  - 컴포넌트 크기 조정으로 더 나은 UX

### 🐛 Fixed
- 카테고리 삭제 기능 복원
- 카테고리 편집 시 상태 관리 개선

## v16 (2024-11-13)

### ✨ Initial Features
- **플립북 카드 형식** - 시각적 AAC 의사소통
- **외부 이미지 URL 지원** - 웹 이미지 직접 사용
- **카드 관리** - 추가, 삭제, 수정 기능
- **한국어 TTS** - ResponsiveVoice API 음성 읽기
- **카테고리 시스템** - 음식, 감정, 인사, 요청, 일상 등
- **즐겨찾기** - 자주 쓰는 카드/문장 저장
- **보조 단어 버튼** - 조사, 동사, 형용사 빠른 추가
- **인포그래픽 그리드** - 7x10 그리드 (70개 셀)
- **오프라인 작동** - localStorage 활용
- **반응형 디자인** - 모바일, 태블릿, 데스크톱 지원
- **CSV 가져오기/내보내기** - 데이터 백업 및 복원
